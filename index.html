常见问题

1. 数据会在 iPhone / iPad 间同步吗？
如果你在系统设置中为本应用开启了 iCloud，同一 Apple ID 下的设备可通过 iCloud 进行同步。首次同步可能需要一些时间。

2. 我的图片/文字会上传到公开网络或开发者服务器吗？
不会。本应用不会将你保存的图片或文字内容上传到开发者自建服务器。你手动使用系统“分享/导出”功能时，分享对象由你自行选择。

3. 如何导出？
在应用内进入导出功能页面，选择需要导出的内容后即可导出（例如导出为 PDF 或通过系统分享）。

4. 为什么换手机/重装后看不到数据？
请确认：
- 使用的是同一个 Apple ID
- 系统 iCloud 已开启，并允许本应用使用 iCloud
- 给同步一些时间完成（尤其首次同步或数据较多时）

5. 如何反馈问题以便更快定位？
建议在邮件中附上：设备型号、系统版本、App 版本号（设置页/关于页可查看）、问题发生步骤与截图。
